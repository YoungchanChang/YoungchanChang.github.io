# 데이터 전처리

### 데이터 전처리란?

- 분석에 적합하게 데이터를 가공하는 작업

### 왜 데이터 전처리가 중요한가

- 데이터 퀄리티가 분석의 결과 중 90% 이상을 좌우한다고 해도 과언이 아니다.

### 데이터 전처리시 주의할 점

1.데이터가 불완전한 경우 : null값이 들어간 경우

2.잡음이 섞여있는 경우 : 주식가격에 -값이 있는 경우

3.모순된 데이터가 있는 경우 : 남성 주민등록번호에 2로 시작하는 경우
