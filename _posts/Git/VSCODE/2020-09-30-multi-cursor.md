# VScode

- vscode에서 여러 단어를 선택해야 하는 경우들이 있다.

### 같은 단어들 선택하기

- Ctrl + D를 누르면 동일한 단어들을 순차적으로 선택할 수 있다.

- Ctrl + Shift + L을 누르면 동일한 단어를 모두 선택할 수 있다.

### 다른 단어들 선택하기

- Alt + Click으로 누른 뒤에 shift + Home으로 여러 일렬로 된 단어들을 선택할 수 있다.

- Ctrl + Alt로 상하 위치 추가할 수 있다.
