# 기타 정보들

### 주석을 쓰는 방법

- 한 줄을 쓸 때는 주석의 줄 앞에 `//`를 쓴다

- 여러 줄을 작성 할 때는 `/* 여러 줄의 내용 */`로 감싼다.

### 줄바꿈과 여백

- 세미콜론으로 **명령이 끝났다를 표시하는 기호**이다.

- 세미콜론을 삭제해도 명령은 잘 동작한다. 줄이 바뀌면 명령이 끝난다라고 간주한다.

- 세미콜론을 적는 것은 좋은 습관이다. 한 줄에 두 명령어를 적을 때는 세미콜론이 필수이다.

```javascript
var a = 1
alert(1)

var a = 1;
alert(1);
```
