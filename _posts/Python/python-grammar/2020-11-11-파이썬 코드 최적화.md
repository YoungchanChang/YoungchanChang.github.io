# 간단한 코드

- 메서드마다 시작시간과 끝시간을 출력해서 한다.

```python
import time
start = time.time()  # 시작 시간 저장
 
 
print("time :", time.time() - start)  # 현재시각 - 시작시간 = 실행 시간
```
