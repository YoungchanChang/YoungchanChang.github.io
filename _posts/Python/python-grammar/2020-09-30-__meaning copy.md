# __뜻__

- 파이썬에서 __를 쓰는 이유는 파이썬자체에서 개발한 함수라는 뜻이다.

# dir()내장함수

- dir()내장 함수는 어떤 객체를 인자로 넣어주면 해당 객체가 어떤 변수와 메소드를 갖고 있는지 나열해준다.

- dir()는 directory 경로라는 의미로, 속성과 변수의 리스팅을 보여준다. 사전처럼 볼 수 있다.
