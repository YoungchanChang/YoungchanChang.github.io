# Pycharm

### 파이참을 쓰는 이유

- vscode에서 설치해서 운영했을 때, Ctrl + 클릭 등으로 자세히 보기 기능이 지원되지 않았다.

- 변수명 자세히 보기와 디버그 기능의 편리함으로 파이참을 쓰기로 결정했다.

### vscode처럼 환경변수처리하기

- 우측 하단을 클릭해서 VirtualEnv에서 New Environment로 만든다.

- 파이참은 키면 아래와 같이뜬다.
