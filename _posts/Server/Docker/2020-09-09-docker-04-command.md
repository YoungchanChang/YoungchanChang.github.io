# Docker


### 도커 상태관리

- 도커 : 이미지를 다운로드 받아서 실행한다.

- 서버의 생성, 배포가 용이하다.

```
docker ps -a

# 도커에서 다운받은 이미지 확인. 이미지 없으면 다운로드를 받는다. 다운로드 받은 것이 PC에 저장된다.
# .iso게임파일처럼 하면 된다.
docker images
```

- 게임 시디를 가져와서 돌리듯이 진행한다.

- 가상머신에서 게임 시디를 돌린다.
